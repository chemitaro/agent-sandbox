# Suggested commands

- Setup/init: `make init`
- Start container: `make start` or `make up`
- Shell into container: `make shell` / `make shell-sandbox`
- Status/logs: `make status`, `make logs`
- Stop/remove: `make down`, `make restart`, `make rebuild`
- Pre-commit: `make pre-commit-install`
- CLI tools install/verify (from package.json): `npm run install-global`, `npm run verify`
